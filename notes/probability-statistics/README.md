# Probability and Statistics Notes

Goal: understand generative models as problems about probability distributions learned from finite data.

## Main topics

1. Probability measures and densities.
2. Empirical measures.
3. Weak convergence and Wasserstein convergence.
4. Statistical/generalization error.
5. Population vs empirical objectives.

## Minimum mastery checklist

- [ ] Can define a probability measure.
- [ ] Can explain empirical distribution.
- [ ] Can distinguish convergence in probability, distribution, and weak convergence of measures.
- [ ] Can explain why generative modeling is distribution learning.
- [ ] Can state a basic law of large numbers argument for empirical risk.
