# Diffusion and Flow Notes

Goal: understand the mathematical foundations of diffusion models, score matching, flow matching, and Schrödinger bridges.

## Main topics

1. Score matching and denoising score matching.
2. Diffusion SDE and reverse-time SDE.
3. Probability flow ODE.
4. Flow matching and conditional flow matching.
5. Schrödinger bridge and entropic OT.

## Minimum mastery checklist

- [ ] Can define the score function.
- [ ] Can explain denoising score matching.
- [ ] Can write the forward and reverse SDE.
- [ ] Can explain the continuity equation for flow matching.
- [ ] Can code toy score matching and toy flow matching.
