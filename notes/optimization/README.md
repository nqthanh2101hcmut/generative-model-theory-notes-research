# Optimization Notes

Goal: build the mathematical foundation for optimization on probability spaces and generative model alignment.

## Main topics

1. Convex optimization.
2. KKT and Lagrangian duality.
3. Mirror descent, dual averaging, primal-dual algorithms.
4. Entropy/KL-regularized distributional optimization.

## Minimum mastery checklist

- [ ] Can state and prove KKT conditions for a simple convex problem.
- [ ] Can derive Fenchel conjugates for common functions.
- [ ] Can explain why KL regularization gives a Gibbs-type solution.
- [ ] Can code exponentiated-gradient/mirror descent on the simplex.
- [ ] Can write the Lagrangian for constrained distributional optimization.
